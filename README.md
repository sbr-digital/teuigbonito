## 📱 Sobre o Projeto

Landing page moderna e responsiva para a agência **Teu IG Bonito**, especializada em marketing digital e gestão de Instagram. O projeto foi desenvolvido com foco em performance, SEO e experiência do usuário.

### ✨ Características

- 🚀 **React 18** com TypeScript para type-safety
- ⚡ **Vite** para build ultrarrápido
- 🎨 **Design moderno** com tema dark purple
- 📱 **Responsive** - Mobile-first approach
- 🔍 **SEO otimizado** com meta tags completas
- ♿ **Acessível** seguindo boas práticas WCAG
- 🎭 **Animações suaves** com intersection observer
- 💬 **Integração WhatsApp** no formulário de contato

## 🛠️ Tecnologias Utilizadas

- **React 18.2.0** - Biblioteca UI
- **TypeScript 5.2.2** - Superset JavaScript tipado
- **Vite 5.2.0** - Build tool e dev server
- **CSS3** - Estilização com variáveis CSS e animações

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/sbr-digital/teuigbonito.git

# Entre na pasta do projeto
cd teuigbonito

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 🚀 Scripts Disponíveis

```bash
# Desenvolvimento
npm run dev          # Inicia servidor de desenvolvimento na porta 3000

# Build
npm run build        # Compila TypeScript e gera build de produção

# Preview
npm run preview      # Preview do build de produção

# Linting
npm run lint         # Verifica código com ESLint
```

## 🎨 Seções da Landing Page

1. **Hero** - Apresentação com logo e CTA principal
2. **Stats** - Estatísticas de resultados (300K+ faturamento)
3. **Serviços** - 6 serviços principais da agência
4. **Portfólio** - Cases de sucesso
5. **Depoimentos** - Feedback de clientes
6. **CTA** - Call-to-action para consultoria gratuita
7. **Contato** - Formulário com integração WhatsApp
