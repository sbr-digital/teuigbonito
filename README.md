# 🎨 Teu IG Bonito - Landing Page

Agência de Marketing Digital especializada em Instagram e redes sociais.

## 🚀 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite 5
- **Language**: TypeScript
- **Styling**: CSS Modules + CSS Variables
- **Deployment**: Vercel

## 📦 Instalação

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build
npm run preview
